# Proyecto de Investigacion de Operaciones I

Aplicacion web interactiva para resolver ejercicios de optimizacion. El proyecto permite trabajar con transporte iniciando por Esquina Noroeste y optimizando con MODI, ademas del metodo Simplex para problemas de minimizacion con restricciones `>=` y maximizacion con restricciones `<=`. La interfaz genera tablas dinamicas, permite cargar ejemplos, muestra el procedimiento paso a paso y presenta los resultados finales de forma clara para apoyar el analisis academico.

## Metodos incluidos

- Metodo de transporte por Esquina Noroeste y optimizacion por MODI.
- Metodo Simplex de minimizacion.
- Metodo Simplex de maximizacion.
