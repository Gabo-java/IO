# Proyecto de Investigacion de Operaciones I

Aplicacion web interactiva para resolver ejercicios basicos de optimizacion vistos en Investigacion de Operaciones I. El proyecto permite trabajar con dos metodos: el metodo de transporte por Esquina Noroeste y el metodo Simplex para problemas de minimizacion con restricciones de tipo `>=`. La interfaz genera tablas dinamicas, permite cargar ejemplos, muestra el procedimiento paso a paso y presenta los resultados finales de forma clara para apoyar el analisis academico.

## Metodos incluidos

- Metodo de transporte por Esquina Noroeste.
- Metodo Simplex de minimizacion sin Gran M ni dual.
